Petal

A new, clean and modern CSS UI framework.

Includes normalize.css by Nicolas Gallagher & Jonathan Neal (MIT License). Uses Lato font by Łukasz Dziedzic (SIL OFL 1.1) as the default font, hosted on Google Fonts.

Petal Grid System

Easily create column layouts for the whole page or parts of the page.

.col-1
.col-2
.col-2
.col-3
.col-3
.col-3
.col-4
.col-4
.col-4
.col-4

Insert .col-x class to whatever divs you want to make columns of, where x is the number of columns the row is divided into. (Currently supports 2,3,4 columns)

Requirements

You must have a single .container div at root level, and a .row element contain the divs you want to group as columns.


Spanned columns

.col-3.colspan-2
.col-3.colspan-2
.col-4.colspan-2
.col-4
.col-4
.col-4
.col-4.colspan-2

Add .colspan-x to column divs, where x is the number of columns you would 'merge' (think of a table layout).


Nesting columns

.col-3
.col-2
.col-2
.col-3
.col-3
.col-3
.col-3
.col-3
.col-1

You can nest any columns inside columns. Just don't forget to nest .row containers as well.

Quick Margins & Paddings

Use quick preset classes to add margins and paddings to any elements.

Margins
  • .mt-10 : 10px top margin
  • .mt-20 : 20px top margin
  • .mb-10 : 10px bottom margin
  • .mb-20 : 20px bottom margin
Paddings
  • .pt-10 : 10px top padding
  • .pt-20 : 20px top padding
  • .pb-10 : 10px bottop padding
  • .pb-20 : 20px bottom padding

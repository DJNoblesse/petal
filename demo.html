Petal

A new, clean and modern CSS UI framework.

Includes normalize.css by Nicolas Gallagher & Jonathan Neal (MIT License). Uses Lato font by ≈Åukasz Dziedzic (SIL OFL 1.1) as the default font, hosted on Google Fonts.

Layout

Base structure

Set a container with .container class in which you will place .rows. Use appropriate HTML5 semantic tags such as <section>,<article>,<header> and <footer> along with the class.

You must have .row sub-containers for child elements to align to the page width.

Container widths and responsive layout

Container width is set as a LESS variable @container-width in the base petal.less file. Default value is 980px; modify the value to change fixed width.

Responsive layout is enabled by default, and the page will automatically fit the window width when screen width is smaller than the container width.

Quick Margins & Paddings

Use quick preset classes to add margins and paddings to any elements.

Margins
  • .mt-10 : 10px top margin
  • .mt-20 : 20px top margin
  • .mt-50 : 50px top margin
  • .mb-10 : 10px bottom margin
  • .mb-20 : 20px bottom margin
  • .mb-50 : 50px bottom margin
Paddings
  • .pt-10 : 10px top padding
  • .pt-20 : 20px top padding
  • .pt-50 : 50px top padding
  • .pb-10 : 10px bottop padding
  • .pb-20 : 20px bottom padding
  • .pb-50 : 50px bottom padding
